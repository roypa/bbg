Here you can see an example of embedding the Flip in a common html page. The flash object is 750x500 pixels and is included via SWFObject inside a container div. The container div has CSS styles to set borders and margins.

To add images edit the xml file provided with this example. In order to change the background to a color other that white you should change it both in the xml file and in the html code used to embed the flip. There are many options that you can set via xml file, refer to www.flshow.net for a complete list.

This slideshow requires Adobe Flash Player 10.0 (or higher). JavaScript must be enabled.
